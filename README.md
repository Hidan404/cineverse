# Cineverse

Descubra filmes, explore categorias e encontre seu próximo filme favorito.

## Setup local

1. Copie `js/config.example.js` para `js/config.js`
2. Obtenha sua chave da API em https://www.omdbapi.com/apikey
3. Substitua `"SUA_CHAVE_AQUI"` em `js/config.js` pela sua chave
4. Abra `index.html` no navegador
